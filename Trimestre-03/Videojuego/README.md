# Ejercicio Videojuego

> Autor: Alejandro del Valle Vallés

El ejercicio cuenta de 3 estados:
1. Inicio: Donde se selecciona la dificultad y se lanza el juego cuando se pulsa "Jugar". Durante este estado se reproduce en bucle una canción.
2. El juego: En base a la dificultad elegida las formas cambian de tamaño siendo más pequeñas o grandes, y el tiempo disponible para pulsarlas.
3. Fin del Juego: Lanzada cuando el jugador pierde todas las vidas. Muestra el puntaje y un botón que envía de vuelta al inicio.

## El Juego
El juego cuenta con 4 modos (Facil, Normal, Dicifil e Imposible). El modo dificil hace que las figuras se muevan, y el imposible solo tiene una vida, mientras que en el resto se consta de 3.
Durante el juego se muestra un contador con el tiempo restante para pulsar la figura que se muestra en pantalla y aparte el numero de vidas disponibles.
También se muestra el puntaje obtenido.
El sistema de puntaje se basa en el tiempo que tarda el jugador en pulsar sobre la figura desde que aparece en pantalla hasta que la pulsa.
