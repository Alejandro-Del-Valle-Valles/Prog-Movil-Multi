# Prog-Movil-Multi
---
> Repositorio del módulo de Programación Móvil y Multimedia

Repositorio con los proyectos de clase para el Desarrollo Movil y Multimedia de forma nativa con Kotlin en Android mediante Android Studio.
El repositorio cuenta con pequeños proyectos a modo de prueba y aprendizaje del diseño y desarrollo de apps mediante Kotlin y XML para Android 8.0 y superior.
Cuenta con diversos proyectos que tratan diversas secciones.

## Trimestre 01:
### Tema 01:
- Creación de proyectos en Android Studio.
- Creación de elementos básicos en XML.
- Manejo básico de elementos desde código.
### Tema 02:
- Creación de multiples Activitys y navegación entre ellas.
### Tema 03:
- Creación de layouts.
- Expansión de creción de componentes.
- Listados y Menús
### Tema 04:
- Notificaciones
- Dialogs
- CardsViews (Avisos en pantalla)
### Tema 05:
- Almacenamiento de ficheros.
- Manejo de JSON y BBDD.
- Consumo de APIs.
### Tema 06:
- Creación de vistas y componentes dinámicas.
